bilibili接口

bilibili接口 是根目录下的 bili-live.js。它是一个Node.js的库。

它只是为了助手而设计的，并没有完善好API，只准备了必需的API。

开始使用

  • 要启用 bilibili接口 很简单，只需要你 : 
    var Bili_live = require('./bili-live.js');
    下面会详细讲解bilibili接口的一些命令。

命令

getLivePageInfo(liveid)

  • liveid bilibili直播间的roomid。
    callback (error , info ) 回调函数。
  • 返回bilibili直播间的基础信息。
  • info:
    sch_id bilibili直播间的当前节目的sch_id。
    meta_id bilibili直播间的当前节目的meta_id。
    state bilibili直播间的当前节目的状态。 PREPARE是准备,LIVE是直播中,END是直播结束。

getLiveInfo(liveid)

  • liveid bilibili直播间的roomid。
    callback (error , info ) 回调函数。
  • 返回bilibili直播间的节目表。
  • info:
    sch_id bilibili直播间的节目的sch_id。
    meta_id bilibili直播间的节目的meta_id。
    start bilibili直播间的节目的开始时间(UnixTiemstamp)。
    start_at bilibili直播间的节目的开始时间(Date)。
    aid bilibili直播间的节目的aid。
    cid bilibili直播间的节目的cid。
    title bilibili直播间的节目的标题。
    description bilibili直播间的节目的描述。
    typeid bilibili直播间的节目的节目类型。
    tag bilibili直播间的节目的关键字。
    state bilibili直播间的节目的状态（不一定准确，以getLivePageInfo为准）。 PREPARE是准备,LIVE是直播中,END是直播结束。
    check_status bilibili直播间的节目的认证状态。

getPlayerInfo(cid,[cookies],callback)

  • cid bilibili直播间节目的cid。
    cookies 登录账号的Cookies(可空)。
    callback (error , info ) 回调函数。
  • 返回bilibili直播间的player信息。
  • info:
    是一个XMLreader类型的变量。具体使用方式请https://www.npmjs.org/package/xmlreader

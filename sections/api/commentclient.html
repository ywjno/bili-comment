弹幕服务器客户端API

弹幕服务器客户端 是 Bilibili 直播弹幕助手的 核心程序。

它可以访问弹幕服务器，同步获取弹幕聊天室里的弹幕、礼物和其它信息。

开始使用

  • 要启用 弹幕服务器客户端 很简单，只需要你 : 
    var CommentClient = require('bili-comment').CommentClient; //bili-comment 是助手的根目录位置
var nowClient = new CommentClient();
    下面会详细讲解Client的一些命令和基础事件。

Class : CommentClient

new CommentClient()

  • 返回一个新的弹幕服务器客户端。

命令行参数

  • -c, --config 弹幕服务器配置文件位置
    如果是通过配置文件启动, 则必须带上这个参数! 优先级最高.
  • -i, --live_id <id>
    和 直播间URL 二选一.
  • -u, --live_url <url> 直播间URL
    和 直播间ID 二选一.
  • -t, --show_time 显示时间
    显示弹幕或者礼物发送的时间
  • -U, --show_username 显示用户名
    显示弹幕发送者的用户名
  • -w, --show_watcher_number 显示观众数目
  • -g, --show_gift 显示礼物
  • -o, --output 是否输出保存弹幕数据
  • -p, --output_path <dir> 保存弹幕数据的文件路径
    默认是 ./comment
  • -f, --output_filename <filename> 保存弹幕数据的文件名
    默认是 %t.log.
    格式化参数: %t == unix timestamp
  • -h, --host <host> 弹幕服务器主机名
    建议不要修改.
  • -p, --port <port> 弹幕服务器端口
    建议不要修改.
  • --sver <version> 弹幕服务器版本
    默认是最新版, 建议不要修改.

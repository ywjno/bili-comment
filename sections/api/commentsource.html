弹幕源数据

弹幕源数据 是根目录下的 comments文件夹内的 .source文件。

它是为了助手而设计的，所以开发读取.source文件需要看一下下面的开发文档。

文件结构

  • [live_programinfo,0x00,0x00,stream1,0x00,stream2,0x00,...,streamN,0x00]

数据格式

live_programinfo

  • 请看bilibili接口-命令-getLiveInfo-info

Stream newCommentString

  • 请看弹幕服务器客户端-Class : Client-Event 'newCommentString'-源数据格式

Stream watcherNum

  • action Stream 动作类型
    num 弹幕聊天室在线人数
  • 弹幕聊天室观众人数

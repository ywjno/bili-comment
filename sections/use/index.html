安装和使用

安装

Bilibili 直播弹幕助手 是纯绿色软件。只要正确安装了Node.js，助手是直接解压就能用的.

使用

在Window之下运行 Bilibili 直播弹幕助手 其实很简单，只要双击 run.cmd 就可以了，是不是很简单。

如果是其他系统，在助手的根目录下运行 node bili-comment.js 这个代码就OK了。

刚刚解压好了 Bilibili 直播弹幕助手 并非意味着安装结束了。还要设置直播间!

有关 Bilibili 直播弹幕助手 设置直播间的问题，请移步 快速配置工具 在线生成配置文件。

我有特殊的使用技巧

想让助手窗口没那么宽又可以全部内容都显示吗？
只要右键单击 助手的cmd标题栏 -> 属性(P) -> 布局 -> 屏幕缓冲区大小 -> 宽度 调小一点，调到适合 就可以了。

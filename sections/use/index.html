安装和使用

安装

Bilibili 直播弹幕助手 是纯绿色软件。只要正确安装了Node.js，助手是直接解压就能用的.

初次配置

刚刚解压好了 Bilibili 直播弹幕助手 并非意味着安装结束了。还要设置直播间!

现在的Bilibili 直播弹幕助手支持两种启动的方式。第一种是 带命令行参数运行，第二种是 通过读取配置文件运行。

前者为 专家模式。启动传参类似于： node cli -i 19386 -U -g -o。

专家们可以运行 node cli -h 查看支持的参数，下面不再阐述这种方法的启动方式

后者为简单的方法，可以通过网页端生成的配置文件 或者 利用命令行生成配置文件（查看帮助: node configBuilder -h）来启动助手。

下载配置文件（好比说保存为 config.json），保存在 Bilibili 直播弹幕助手的根目录上。

Windows 系统

如果你是 Windows，可以尝试使用 .cmd启动助手。

只需要改我们提供的 run.cmd (在助手的根目录)， 把里面的内容配置成 node cli -c config.json（config.json是配置文件路径）。

其它 系统

可以尝试使用 bash 写个脚本运行 node cli -c config.json

运行

Windows系统的主播只需要双击 run.cmd 就可以了启动了~~是不是很简单。

如果是其他系统，执行你们写好的脚本或者直接运行程序都OK。

我有特殊的使用技巧 (限Windows)

想让助手窗口没那么宽又可以全部内容都显示吗？

只要右键单击 助手的cmd标题栏 -> 属性(P) -> 布局 -> 屏幕缓冲区大小 -> 宽度 调小一点，调到适合 就可以了。
